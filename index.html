Covid_19 Case Analysis Using Python

This is a kaggle dataset project python on Jupyter notebook using Numpy, Pandas, Seaborn and Matplotlib.

This project anaylzes a new coronavirus designated 2019-nCoV was first identified in Wuhan, the capital of China's Hubei province.

This project reads covid_19 analysis(country wise latest) from an Excel file, processes it, and then generates visualizations to analyze the data.
